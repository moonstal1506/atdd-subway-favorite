- [x] 개인별로 관리할 수 있는 즐겨 찾기
- [x] 인증 기반의 인수 테스트 작성 시 인증 정보는 재사용
- [x] 즐겨찾기 생성
- [x] 즐겨찾기 조회
- [x] 즐겨찾기 삭제
- [x] 예외사항

      - [x] 내 정보 관리 / 즐겨 찾기 기능은 로그인 된 상태에서만 가능

- [x] 즐겨찾기 삭제 시 토큰이 없는 경우에 대한 요청 테스트
- [x] 즐겨찾기 삭제 시 본인이 아닌 경우에 대한 요청 테스트
